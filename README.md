# AI Gesture with voice Media Controller 

Control video playback and screen scrolling using **hand gestures and voice commands**.

## Features (Planned)
- Hand movement to Scroll page (up/down/left/right)
- Voice commands to Play / Pause media
- Real-time computer vision using MediaPipe
- Automation using PyAutoGUI

## Tech Stack
Python, OpenCV, MediaPipe, PyAutoGUI, SpeechRecognition

## Development Roadmap
1. Webcam capture
2. Hand tracking
3. Gesture direction detection
4. Scroll automation
5. Voice command integration
6. Final integration & demo
